# Security Policy

## Supported Versions

We recommend using the latest version of SP for the best security and performance.

| Version | Supported          |
| ------- | ------------------ |
| v0.1.4  | :white_check_mark: |
| < 0.1.4 | :x:                |

## Reporting a Vulnerability

We take security seriously! If you find a potential vulnerability, please **DO NOT** open a public issue. Instead, report it privately to:

**nikolamatic225@gmail.com**

Please include:
1.  **A description** of the vulnerability.
2.  **Reproduction steps** (ideally an `.sp` script).
3.  **Your platform** (OS and version).

We will aim to respond within 48 hours and work with you to resolve the issue as quickly as possible.
