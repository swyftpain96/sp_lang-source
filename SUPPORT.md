# Support Guide

Welcome to the SP Programming Language support page! We're here to help you get the most out of your coding experience.

## 🆘 How to Get Help

### 1. **Check the Documentation**
Before reaching out, please check the following:
- **[Language Guide](./LANGUAGE_GUIDE.md)**: Syntax, built-in functions, and more.
- **Example Scripts**: Look for `.sp` files in the repository.

### 2. **GitHub Issues**
If you suspect you've found a bug or have a suggestion, feel free to open an issue:
- **[Bug Report Form](https://github.com/mrmrcl/sp_lang/issues/new?template=bug_report.yml)**
- **[Feature Request Form](https://github.com/mrmrcl/sp_lang/issues/new?template=feature_request.yml)**

### 3. **Community & Chat**
- **Discord**: Join the community! You can find **SwyftPain** in the server.
- **Discussions**: We plan to enable GitHub Discussions soon for broader community Q&A.

### 4. **Direct Contact**
For critical issues, security vulnerabilities, or other specific inquiries:
- **Email**: **nikolamatic225@gmail.com**

## 🤝 Contributing
